Software Used In Class

Ableton Live allows you to record and mix music, manipulate sounds, and play as an instrument during a live performance. It is a very versitile program for sound design and circuit bending.

The Adobe Suite is offered for free to Digital Culture students, meaning you get full access to Photoshop, Illustrator, InDesign, and more at no cost to you. The Adobe Suite deals with digital creation of multimedia, whether it's through animation, alteration of photos, or digital drawings.

Arduino offers both a digital circuitboard drafting service where you can assemble and test your creations virtually and physical components you can purchase to build out your idea. Arduino offers both hardware and software components, in which circuits are rigged to a microcontroller that can feed information to your computer. Through programming and components you add to the circuit, such as LEDs or buttons, you can create personalized autonomous and interactive circuits to solve problems or entertain your audeince.

Android Studio is compatable for Google's official Android operating system. Using this program, you can develop apps for Android use. Whether you have a Windows, Mac, or Linux, Android Studio can be downloaded to create your app!

Blender is a free open-source platform for creating and rendering virtual 3D objects. This program could be used for animation, rigging, design, and visual effects. The user interface can be altered using Python as the coding language.

Final Cut Pro allows for non-linear video editing, in which the user can easily rearrange, delete or add video footage. The final video can be exported in a variety of file formats, allowing for versatility.

Fusion 360 is a 3D computer aided design software program that allows users to render, design, create, and test products and circuit boards used for engineering, design, and manufacturing purposes. This program can be downloaded on Windows, Mac, or as a simpler version on Apple and Android phones.

GIMP is a free open-source image manipulation software that can be used for drawing, image editing and manipulation, and changing the file format of various images. This versatile progam can be used on Windows, Mac, or Linux devices.

Logic Pro is a professional sound design software that is extremely robust and versitile. Users can record, edit, change instruments, import sound or video, and manipulate sounds to create audio or scores for projects. This software is compatable with Mac devices.

Max MSP is a very versitile and unique software that allows users to create interactive art, sound, or customizable creation through visually-oriented programming. Through an extensive and complex list of functions and plug ins, the possibilities of creation with Max are virtually endless. The freedom for creative planning is a keystone in Digital Culturre, thus making Max the perrfect candidate to represent the personalization of this major as a whole.

Maya is a realistic 3D rendering and desining software that allows users to develop characters and assests for scenes in movies, TV shows, animated shows, and designs. It is compatable with Mac, Windows, and Linux products.

Processing allows coders to draft their ideas and access a large library of code. This program aims to bridge coding and the arts together, hosting p5.js, a visual coding platform. This program can be hosted on any device that can connect to the internet.

Raspberry Pi is a small, cost-efficient computer that looks like a circuitboard that programmers can use to learn computational thinking, computer science skills, and coding in Python and Scratch languages. It can be used in addition to or in place of certain computational devices.

Rhino is a 3D modeling and design software that can be used to create art, commercial models, and other functional objects. This program can be used with Mac and Windows decives.

TouchDesigner is an accessable visual coding platform capable of producing interactive projections, realistic 3D simulations, and real-time multimedia content. This program can be used with Mac and Windows devices.

Unity is a massive cross-platform game engine capable of rendering 3D environments and objects for entertainment, manufacturing, and advertising. Unity uses the coding languages C++ and C#. This program is compatable with Mac and Windows devices.

Unreal Engine is a 3D design tool for rendering realistic objects and environments in games, videos, and other creative ambitions. Unreal Engine uses the coding languages C++ and Blueprint. This program is compatable with Windows, Mac, and Linux devices.

Visual Studio Code is a code editor, debugger, and tester for HTML, CSS, JavaScript, and JSON. It can also be adjusted to include C++, PHP, or Python coding languages. This program is compatable with Windows, Mac, and Linux devices.
